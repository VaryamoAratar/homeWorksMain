# Задача 3. Вывод структуры

## Описание
Создаем структуру для хранения адреса.
Структура содержит название города, улицы, номер дома, номер квартиры
и индекс.

Создаем функцию для вывода структуры на консоль.

Тестируем созданную функцию. Создаем в коде экземпляры структуры,
передаем их в функцию и выводим на консоль получившиеся экземпляры
структуры.
