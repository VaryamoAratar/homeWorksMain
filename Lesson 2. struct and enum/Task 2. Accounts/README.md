# Задача 2. Счета

## Описание

* Создаем структуру для хранения информации о банковском счёте.
Она хранит номер счёта (целое число), имя владельца (строка) и
количество денег на счету (дробное число).

* Создаем функцию для изменения баланса счёта.
Функция должна принимать экземпляр структуры банковского счёта и новую сумму.

* Пользователь заполняет поля экземпляра структуры: -> "Новый баланс счёта" 

Выводим обновлённый экземпляр структуры на консоль.
